Box Model

The CSS Box Model describes how every HTML element is structured like a rectangular box on a web page.
Each element has four main layers:

  • Content: The actual text or image inside the box.
    Syntax:

    Hello World

  • Padding: Space between content and thr border.
    Syntax: padding: 20px;
  • Border: The visible line surrounding the padding .
    Syntax: border: 3px solid blue;
  • Margin: Space outside the border (between this box and others).
    Syntax: margin:15px:

Background in CSS

The background property in CSS allows you to add:
  • Solid colors
    Syntax: background-color: lightblue;
  • Gradient colors (smooth color blends)
    Syntax: background: linear-gradient(to right, red,yellow);
  • Images (for background images)
  • rgba() Adds color + transparency
    Syntax: background-color: rgba(0, 0, 255, 0.3);

Border

The border is a line drawn around the padding and content of an element. It is part of the box model and takes up space in the layout.
Syntax: border: 2px solid blue;

outline

The outline is a line drawn outside the element‚Äôs border. It is not part of the box model, so it does not take up space ‚Äî it just highlights the element.
Syntax : outline: 2px dashed red;

Fixed Width / Height

Fixed size means the element‚Äôs width and height stay the same no matter the screen size (not responsive).
Syntax: .box-fixed { width: 300px;
height: 200px; }

Relative Width / Height

Relative size changes according to the parent element or screen size ‚Äî more responsive.
Syntax: .box-relative { width: 80%; height: auto;}
